# 🏋️ CardioCrew

[![Live Demo](https://img.shields.io/badge/live%20demo-cardiocrew.io-brightgreen)](https://banksiasprings.github.io/CardioCrew/)

A vibrant fitness challenge Progressive Web App where users join clans, complete daily workout challenges, earn points, and compete on global leaderboards. Built with Firebase for real-time sync and push notifications.

## ✨ Features

- **Daily Challenges** – 4 daily workout challenges + bonus challenges
- **Clan System** – Create or join clans to compete as a team
- **Leaderboards** – Real-time global and clan-based rankings
- **Wagers** – Stake points on challenge completions
- **Push Notifications** – Get reminded of new challenges
- **Multi-Campaign** – Multiple fitness campaigns with different themes
- **Offline Support** – PWA with service worker caching
- **Responsive Design** – Works on desktop, tablet, and mobile

## 🛠️ Tech Stack

- **Frontend:** Vanilla JavaScript (ES6+)
- **Backend:** Firebase (Firestore + Authentication)
- **PWA:** Service Worker for offline functionality
- **Hosting:** GitHub Pages
- **Version:** v16

## 🚀 Getting Started

1. **Visit the live app:** https://banksiasprings.github.io/CardioCrew/
2. **Create an account** with email or social login
3. **Join a clan** or create your own
4. **Complete daily challenges** and earn points
5. **Climb the leaderboards** with your clan

No installation required—just open in your browser!

## 📦 Installation (for development)

```bash
git clone https://github.com/banksiasprings/CardioCrew.git
cd CardioCrew
# Serve the root directory with any static file server
python3 -m http.server 8000
# Open http://localhost:8000 in your browser
```
